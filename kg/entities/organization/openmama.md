High-performance, middleware-agnostic messaging API for financial market data —
interfaces with a variety of proprietary and open-source message-oriented middleware
systems. 178 stars, C++. Long-running FINOS project (predates many newer additions),
originally a Linux Foundation effort. Licensed **LGPL-2.1** — a notable exception to
the Apache-2.0 default across most FINOS projects. Project site: https://openmama.org.

Source: https://github.com/finos/OpenMAMA, https://openmama.org
