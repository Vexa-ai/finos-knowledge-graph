AasC, also known as **CALM** (Common Architecture Language Model) — aims to devise
and manage software architecture via a machine-readable, version-controlled codebase,
fostering understanding, efficient development, and maintenance of complex software
architectures. 350 stars, TypeScript, Apache-2.0. Project site: https://calm.finos.org.
[[Matthew Bain]] ([[Morgan Stanley]]) authored CALM's 2026 H1 semi-annual FINOS
project report to the [[Technical Oversight Committee]] (issue #292).

Source: https://github.com/finos/architecture-as-code, https://calm.finos.org,
https://github.com/finos/technical-oversight-committee/issues/292
